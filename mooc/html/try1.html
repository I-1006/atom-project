认识标签1

强调语句strong和em

stong标签默认将需要强调的语句用黑体表示而em标签则讲需要强调的语句用斜体表示

使用span标签为文字单独设置格式

strong\em和span的区别在于: 1.strong和em标签是为了强调一段话中的关键字，语义为强调。 2. 而span标签是没有语义的其作用就是为了设置单独的样式(需要对在head标签内对span进行设置，使用style标签)

q标签，短文本引用

1.引用的句子不需要加双引号，会自动添加。2.标签的重点是它的语义：引用。

blockquote标签,长文本引用

当所需要引用的句子较长时，比如引用一句诗使用blockquore标签。blockquote标签采用缩进句式如

苟利国家生死以，岂因祸福避趋之

br/标签,分行显示文本

该标签相当于enter键，如：
苟利国家生死以，
岂因祸福避趋之。

&|nbsp，空格

在html代码中使用空格没有作用，要想输入空格必须输入  (&|nbsp) 

hr/标签，水平横线

/hr标签和/br标签同样是空标签，只有一个开始没有结束标签

address标签，联系地址信息

网站的联系地址、公司地址、电子邮件地址...等等，这些常用address标签如电子科技啊大学的地址：

西源大道1006号

表单标签与浏览者交互

一级标签form

form成队出现\action浏览者输入的数据被传送到的地方，如一个php页面\method数据传送的方式（get/post）
如form method="传送方式" action="服务器文件"
  1. 所有表单控件（文本框、文本域、按钮单选框、复选框等）必须放在form内
  2. method(get/post)是后端程序员的问题

文本输入框、密码输入框、提交按钮\重置按钮

语法

input type="text/password/submit(仅此有提交作用)/reset(仅此有重置作用)" name="名称" value="文本" /
  1. type标签：
    • type="text"时，输入框为文本输入框。
    • type="password"时，输入框为密码输入框。
  2. name标签：为文本框命名，以备后台程序ASP、PHP使用。
  3. value:为文本输入框设置默认值。（一般起提示作用）

textarea文本域

当用户需要在表单中输入大段文字时，需要用到文本输入域。

语法：

|textarea rows="行数" cols="列数"|文本|/textarea|
  1. textarea标签成对出现。其间可以输入默认值。
  2. cols：多行输入域的列数。
  3. rows：多行输入域的行数。

举例

使用单选框、复选框，让用户选择

设计调查表时使用选择框

语法

input type="radio/checkbox" value="值" name="名称" checked="checked"/
  1. type:
    • 当type="radio"时，控件为单选框。
    • 当type="checkbox"时，控件为复选框。
  2. valuc：提交数据到服务器的值（后台程序php使用）
  3. namc：为控件命名，以备后台程序ASP、PHP使用
  4. checked:当设置checked="checked"时，该选项被默认选中
